2022년 1월 릴리즈 정보

EasyPOS​

연대 생협 이전 내역 매출 재전송 기능 (마감 정산 화면 내 추가)​

CUBOX 안면인식 모듈 연계 고객 조회 기능 추가 (고객조회, 신규고객등록, 고객변경)​

이노템 전자저울 기능추가 - 전자저울 T28 장비 시리얼 연동​

전자메뉴판 주방출력물 출력일련번호 추가 (주방출력물 일반포맷 변경 - 출력물 일련번호)​

콘래드 호텔 발렛 기본요금 및 추가요금 행사가격 자동화 기능. (마감 시 마스터 및 행사 재적용)​

알림톡 호출 기능 추가​

마감데이터 합산 처리 옵션추가 - 특정 포스번호에 대한 마감 데이터를 POS 마감정산 화면에 합산하여 표시 처리​

이지웰페이 기능 개선​

  • 이지웰페이 현금영수증 자진 발급 시 현금 영수증 처리 확인 팝업 미 출력 처리.
  • 인증번호 재차 사용시 오류 메시지 출력 후 다운현상 수정.

특수 가맹점 기능​

아웃백

  • 매해 해당 년도에 대해 CRM 영수증 문구/화면 출력 시 시스템 일자 출력으로 변경 처리

  • CRM 응답전문 오류에 따른 외부고객전표 저장 시 기능 보완 처리.

  • 신규 오퍼코드 추가에 따른 사용 시 분기처리 로직 수정

설빙

  • 포스 매출 제한 기능. - 시스템일자/영업일자 불일치에 대한 포스 매출 제한 기능 추가

탐앤탐스

  • 쿠프마케팅 취소 시 기능 개선. 쿠프마케팅 사용취소에 대한 실제 사용일자/시간 기입 추가

주렁주렁

  • 알림톡 고객전화번호 등록 기능, 알림톡 고객 호출 기능키 및 화면 추가

공차

  • 공차 제휴 할인 시 차액 가감되는 기능으로 발생하는 오류 수정

  • 판매가 변경에 따른 모바일상품권 할인금액 처리

  • 판매가 변경에 따른 모바일 상품권(쿠프마케팅/E-GIFT) 할인 처리 구문 추가

  • 간헐적인 메모리 덤프 문제 예외 처리 추가

  • 쿠프 마케팅 모바일 쿠폰 타입 정보 누락 건 추가

  • 공차 쿠프 마케팅 모바일 쿠폰 사용 시 적립 처리되지 않던 오류 수정

  • 공차 멤버십 쿠폰 사용시에도 적립되는 오류 수정

  • 환경설정-DB조회 화면 개선 처리 (쿼리 히스토리 및 PAD 출력 데이터 짤림 현상 개선)

  • 환경설정-유지보수 화면 개선 처리 (마스터 재생성 간소화 및 '전체마스터다운->개별마스터다운'으로 개선)

오류 수정 사항​

  • 전자메뉴판 에이전트 라이브 체크로직 추가.(전자메뉴판 에이전트 다운 시 재실행 처리 기능)
  • 안드로이드 KDS 연동 시 테이블명 중복 표기오류 개선.
  • 현금영수증 사후발행 부가세 계산 오류 수정
  • 쿠프마케팅 중복 전표 생성 오류 수정. (중복클릭으로 인한 이중 전표 생성 방지)

EasyKIOSK​

쿠프마케팅 교환권 기능 변경​

EasyDID 연동 및 환경설정 화면 개발​

  • 안드로이드 키오스크 연동 및 DID 통신 구조 변경
  • 대기번호 채번 추가(선결제매장)

특수 가맹점 기능​

공차

  • 쿠프마케팅 교환권 적용 시 상품정보 매핑 기준 옵션 처리 (상품코드 or 카테고리 정보로 조회)

  • 명판로고 다운로드 기능 'ASP>명판>매장로고' 정보 다운로드 기능 개발

  • 공차 제휴 할인 시 차액 가감되는 기능으로 발생하는 오류 수정

  • 공차 판매가 변경에 따른 모바일 상품권(쿠프마케팅/E-GIFT) 할인 처리 구문 추가

  • 공차 쿠프 마케팅 모바일 쿠폰 사용 시 적립 처리되지 않던 오류 수정

더벤티

  • 상품 숨김 처리 기능.

  • 터치키 공간 점유/미점유 기능 개발.

  • 추천 메뉴 팝업 화면 추가 (추천메뉴 1개만 선택/등록 가능)

야미매출I/F

  • 고객 핸드폰 번호 필드 추가 기능

오류 수정 사항​

  • 키오스크 일 판매 수량 체크 기능 오류 수정

  • 키오스크 고객 포인트 SMS 인증 오류 수정 (콤파(,)와 포인트(P)값 삽입되어 인증되지 않는 현상)

  • 환경설정에 '영업방식'이라는 구분자가 추가되어 유통매장으로 영업할 수 없던 오류 수정

  • 공차 멤버십 쿠폰 사용시에도 적립되는 오류 수정

  • 일 판매 수량 '사용 안함'과 '서버 조회' 설정 시 품절 여부를 체크하지 않는 오류 수정
